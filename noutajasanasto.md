# Noutajasanasto

Noutajakoirien koulutuksessa ja kokeissa käytettyä sanastoa.

## Kokeet ja lyhenteet

| Lyhenne | Nimi | Selitys |
|---------|------|---------|
| **NOU** | Noutajien taipumuskoe | Ensimmäinen koe noutajan koeuralla. Suoritetaan kylmällä riistalla. Koirille annetaan laatuarvostelut sijoitusten sijaan. |
| **NOME-A** | Noutajien metsästyskoe (lämmin riista) | Metsästyskoe jossa noudetaan tuoreesti ammuttua riistaa. Vaatii metsästysluvan. |
| **NOME-B** | Noutajien metsästyskoe (kylmä riista) | Metsästyskoe jossa noudetaan kylmää riistaa. Luokat: ALO (alokas), AVO (avoin), VOI (voittaja). |
| **WT / NOWT** | Working Test | Noutajien käyttökoe jossa noudetaan dameja. Pisteytetään ja sijoitetaan paremmuusjärjestykseen. |
| **KV-koe** | Kansainvälinen koe | Vastaa NOME-A -koetta, mutta jaetaan erikoispalkintoja. |
| **ALO** | Alokas | NOME-B kokeen aloitustaso. |
| **AVO** | Avoin | NOME-B kokeen keskitaso. |
| **VOI** | Voittaja | NOME-B kokeen ylin taso. |
| **MEJÄ** | Metsästäjäin jälki | Verijälkikoe jossa koira seuraa simuloitua haavoittuneen riistan jälkeä. |

## Koulutuksen perustermit

### Kolme perustehtävää

| Termi | Englanniksi | Selitys |
|-------|-------------|---------|
| **Markkeeraus** | Marking | Koira näkee damin/riistan putoavan ja muistaa paikan. Lähtee noutamaan vasta luvalla. |
| **Ohjaus** | Handling / Blind retrieve | Ohjaaja tietää missä nouto on ja ohjaa koiran siihen pillin ja käsimerkkien avulla. Koira ei ole nähnyt noutoa. |
| **Haku** | Hunt / Hunting | Koira etsii itsenäisesti hakualuetta käyttäen vainuaan. Ohjaaja osoittaa alueen, koira työskentelee vapaasti. |

### Muut koulutustermit

| Termi | Englanniksi | Selitys |
|-------|-------------|---------|
| **Dami / Dummy** | Dummy | Kangaspäällysteinen harjoitusnoudon esine. Eri painoisia ja kokoisia. |
| **Perusasento** | Heel position | Koira istuu ohjaajan vasemmalla puolella, katse eteenpäin. |
| **Palautus** | Delivery | Koira tuo noudon ohjaajalle ja luovuttaa sen käteen. |
| **Luovutus** | Release / Give | Koira päästää irti noudosta ohjaajan käskystä. |
| **Stadga** | Steadiness | Koiran maltti ja itsehillintä - odottaa lupaa ennen lähtöä. |
| **Pillihallinta** | Whistle control | Koiran reagointi pillimerkkeihin. |
| **Linja** | Line / Lining | Koira kulkee suoraan osoitettuun suuntaan. |
| **Blindinouto** | Blind retrieve | Nouto jota koira ei ole nähnyt - ohjataan perille. |
| **Muistinouto** | Memory retrieve | Koira muistaa aiemmin näkemänsä noudon paikan. |
| **Tuplat / Triplat** | Doubles / Triples | Kaksi tai kolme peräkkäistä noutoa samassa tehtävässä. |

## Pillimerkit

| Merkki | Tarkoitus |
|--------|-----------|
| **Yksi pitkä vihelys** | Istu / Pysähdy ja katso ohjaajaa |
| **Useita lyhyitä vihellyksiä** | Tule luokse |
| **Kaksi lyhyttä vihellystä** | Käänny (joissain järjestelmissä) |

## Käsimerkit

| Merkki | Tarkoitus |
|--------|-----------|
| **Käsi ylös** | Mene taaksepäin (back) |
| **Käsi sivulle** | Mene oikealle/vasemmalle (over) |
| **Käsi alas/eteen** | Mene eteenpäin / hae edestä |

## MEJÄ-kokeen sanasto

| Termi | Selitys |
|-------|---------|
| **Veretys** | Jäljen tekeminen verellä. |
| **Lähtö** | Jäljen alkupiste, 30x30 cm potkittu alue. |
| **Kaato** | Jäljen loppupiste, jossa sorkka makaa. |
| **Makaus** | Haavoittuneen riistan makuupaikka jäljellä. |
| **Kulma** | 90 asteen käännös jäljessä. |
| **Katko** | Veretön osuus mestaruustason jäljessä. |
| **Avomerkintä** | Jäljen merkintä näkyvästi maastoon. |
| **Piilomerkintä** | Jäljen merkintä joka ei näy tulosuunnasta. |
| **Maavainuisuus** | Koira seuraa jälkeä nenä maassa. |
| **Ilmavainuisuus** | Koira hakee vainua ilmasta. |
| **Hukka** | Koira hukkaa jäljen eikä löydä sitä itse. |
| **Paluujälki** | Koira palaa jäljellä taaksepäin (virhe). |

## Koiran ominaisuudet

| Termi | Selitys |
|-------|---------|
| **Noutohalu** | Koiran luontainen halu noutaa esineitä. |
| **Hakuinto** | Koiran into etsiä ja löytää. |
| **Riistankäsittely** | Miten koira käsittelee riistaa suussaan. |
| **Vesityöhalu** | Koiran into työskennellä vedessä. |
| **Yhteistyökyky** | Koiran halu työskennellä ohjaajan kanssa. |
| **Itseluottamus** | Koiran varmuus uusissa tilanteissa. |
